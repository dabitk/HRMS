# HRMS
Human Resource Management System developed using Spring Data JPA

회원관리 페이지 주 기능 (2020-04-10 완료):
- 필터를 적용한 검색
- 페이지네이션
- 회원 정보 조회/수정/삭제/신규등록

인사 페이지 주 기능(작업중) :
- 회원 테이블과 인사 테이블 1:1 매핑
- 인사 테이블과 학력 테이블, 인사 테이블과 경력 테이블, 인사 테이블과 자격 테이블 1:N 매핑
- 인사정보 조회/수정/삭제/신규등록
